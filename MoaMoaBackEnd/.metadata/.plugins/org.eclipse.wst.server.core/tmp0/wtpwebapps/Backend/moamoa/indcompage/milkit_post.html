피자 만드는 법

재료: 강력분(200g), 물(120ml), 드라이이스트(4g), 설탕(10g), 소금(4g), 올리브오일(15g)

믹싱 볼에 올리브오일을 제외한 모든 재료를 넣어주세요. 이때, 이스트와 설탕 소금은 엉겨 붙지 않도록 간격을 두고 넣어주세요.

반죽기를 사용할 경우 저속(1단)으로 클린업 단계까지 믹싱해주세요. 반죽기가 없다면 직접 손으로 치대주셔도 좋아요.

올리브오일을 넣고 계속 치대주세요.

반죽 위로 비닐을 덮어준 뒤 따뜻하고 습한 곳에서 40분~1시간 정도 1차 발효시켜주세요.

발효는 오른쪽 사진처럼 반죽이 2배 이상 부풀어 오르고 비닐에 비친 반죽에 구멍이 뽕뽕 보이면 발효가 잘 된 거라고 보시면 돼요.

1차 발효를 마친 반죽은 양손으로 감싸듯이 잡고 둥글리기 해주세요. 다시 비닐을 덮어 실온에서 15분 중간 발효 시켜주세요.

밀대를 이용하여 피자 도우 반죽을 얇게 밀어주세요. 이때 반죽의 크기는 팬의 지름보다 0.5~1cm 정도 더 크게 밀어주세요.

반죽 위로 토마토소스 및 토핑을 수북하게 얹어줄 거라서 구멍이 일정하게 예쁘게 낼 필요는 없어요. 무작위로 콕콕 가볍게 찍어가며 구멍을 만들어주세요.

반죽 위로 피자 토핑을 가득 넣어주세요. 180도 예열된 오븐에서 20분 정도 구워주세요.
